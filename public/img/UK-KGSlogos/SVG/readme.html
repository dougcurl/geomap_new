UK Responsive SVG Lockups

The provided lockup graphic files are Scalable Vector Graphics (SVGs). They can be used just like a JPG or PNG by linking to them with an html img element. Unlike those file types, they will render cleanly at various resolutions and display pixel densities. The lockup SVGs also include media query breakpoints to ensure that the lockup displays correctly. The lockup file will not display the UK Logo less than 70px wide, and also provides a narrow mobile scale of the wordmark element of the lockup.

It is important to note that SVGs are not natively supported by some deprecated browers, such as Internet Explorer 8 or below. See below for fallback code if support for those browsers is required.

Using the Lockup

All public web materials are required to use an official University, unit, or College lockup in the global header, and the University lockup in the footer. Here are some basic points for adding the lockup to your page:

  1. Add your unit/college lockup to the global header, in the upper left corner of your site's content area.
  2. Add the University lockup to your footer.
  3. Give the lockup img elements a CSS min-height of 60px, and a min-width of 260px.

Breakpoints

The lockup file uses the size of its img element to determine which version of the lockup to display. There are 3 versions of the lockup that will display, depending on available space:

Full-size, scaling
If the height of the img is above 60px, and the width of the img is above this lockup's specific minimum full-size width (276px), the lockup will scale up to the width and height provided. The displayed lockup will maintain proper aspect ratio regardless of the width and height provided by CSS.
Mobile
If the width of the img is below the lockup's minimum full-size width (276px), the lockup will compress the wordmark element in order to fit within the minimum mobile width of 260px. The UK logo element will maintain a minimum size of 70px by 60px.
Logo-only Fallback
If the img width is below the minimum required 260px, only the UK logo element will display. NOTE: This is only a fallback, and should be avoided. Be sure to always provide at least 260px width for your lockup.
Hidden
If the img width is below 70px, or the height is below 60px, the lockup will not display at all, in order to prevent incorrect scaling or cropping of the lockup.

PNG fallback

SVGs loaded through HTML img elements are supported on all modern browsers, and in Internet Explorer back to version 9. If you need to support deprecated browsers that lack SVG support, the code below provides a simple in-line fallback.

<img src="lockup.svg" onerror="this.src='lockup.png'"/ >

Other more robust fallback solutions can be found online, including using javascript libraries such as Modernizr.
